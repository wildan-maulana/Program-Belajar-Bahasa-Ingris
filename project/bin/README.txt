Minimum system requirement :
Pentium 4 - 1,4 GHz
RAM 256 MB
Harddisk 4 GB

Cara instalasi :
- Klik 2x file "Setup_Belajar Bahasa Inggris.exe"
- Klik Next
- Klik Next
- Pilih direktori file untuk tempat 
  penyimpanan hasil instalan kemudian klik Next
- Pilih direktori penyimpanan shortcut kemudian klik Next
- Klik "create a dekstop icon jika ingin meletakkan
  shortcut di dekstop kemudian klik Next
- Klik Install
- Klik Finish
- Game sudah bisa dimainkan
